# online-film-website-info-compass
本仓库用于构建在线电影与影视网站的信息指引，对公开可访问的电影站点进行系统化资料整理。收录内容涵盖站点名称、访问地址及站点定位说明，方便集中查看与对照分析。本项目不涉及影视资源的存储、播放或下载，仅作为站点信息档案存在。所有资料来源于公开网络环境，不对第三方站点的内容、合法性或服务质量作出任何评价。
